Menu

Route:
MethodPath
POST /menu

Add a new menu

Authentication

This route requires a valid token cookie set in headers

Body Parameters:
Name Type Description
Menu Object
Menu.name Object Name of menu
Menu.shopId Object (optional) - ShopId link to this menu
Menu.categories Array.<categories> Array of MenuCategory
Menu.categories[x].id string categoryId
Menu.categories[x].products Array.<products> categoryId
Menu.categories[x].products[y].id string productId
Menu.categories[x].products[y].price Number product price
Menu.categories[x].products[y].options Array.<options> (optional) - product options
Menu.categories[x].products[y].options[z].id string option id
Menu.categories[x].products[y].options[z].price Number option price
Response Code:
Type Description
200 if the request is successful
401 if no cookies or malformed cookie
403 if expired cookie
500 if something went wrong
